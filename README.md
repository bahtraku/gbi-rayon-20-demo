# GBI My Home Rayon 20 Prototype
Prototype website profil gereja untuk `GBI My Home Rayon 20` dengan tema `The Year To Arise and Shine`.

## Profil Gereja (Umum)
GBI My Home Rayon 20 adalah komunitas gereja yang berfokus pada pertumbuhan iman, persekutuan, dan pelayanan kasih kepada jemaat serta masyarakat sekitar.

### Identitas Singkat
- **Nama Gereja:** GBI My Home Rayon 20
- **Tema:** The Year To Arise and Shine
- **Lokasi:** Jl. Rawasari Komplek PSMTI No.1, Tanjung Pinang, Kepulauan Riau 29122

### Arah Pelayanan
- Membangun jemaat yang bertumbuh dalam iman dan karakter Kristus.
- Menguatkan persekutuan melalui ibadah raya, persekutuan sel, dan doa bersama.
- Menjangkau masyarakat melalui pelayanan diakonia, sosial, dan penginjilan.
- Membina generasi anak dan pemuda melalui program rohani yang relevan.

### Jadwal Ibadah (di prototype)
- **Minggu:** 08:00 & 10:30 WIB
- **Rabu:** 19:00 WIB
- **Jumat:** 19:00 WIB

## Teknologi yang Digunakan
Project ini dibuat sebagai website statis dengan teknologi:

- **HTML5** untuk struktur halaman
- **Tailwind CSS (CDN)** untuk styling dan utility classes
- **JavaScript (Vanilla JS)** untuk interaksi dasar dan logika UI
- **Alpine.js** untuk state management ringan di sisi frontend
- **Font Awesome** untuk ikon
- **Google Fonts** (`Montserrat`, `Space Mono`, `DM Sans`) untuk tipografi

## Struktur Project
- `index.html` – file utama tampilan website

## Cara Menjalankan
Karena ini project statis, cukup buka `index.html` langsung di browser.

Alternatif lain:
- Jalankan dengan live server dari editor (misalnya VS Code Live Server).
- Atau gunakan static server lokal sesuai kebutuhan.

Dokumen ini disusun untuk kebutuhan profil umum dan pengenalan teknologi pada tahap prototype.

## Font Utama Web
Gotham HTF: Variasi: Bold (untuk judul besar) dan Light/Book (untuk sub-judul atau teks pendukung).
Sentinel: Sering digunakan sebagai font pendamping untuk teks tubuh (body copy) atau kutipan. Sentinel adalah font slab-serif yang dirancang untuk keterbacaan tinggi namun tetap memiliki karakter klasik yang hangat.

### Font Alternatif (Sistem)
Helvetica atau Arial (sebagai pengganti Gotham).
Georgia atau font Serif standar lainnya (sebagai pengganti Sentinel).
